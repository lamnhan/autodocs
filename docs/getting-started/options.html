The Options interface

Custom generator options

Options can be provided in 3 ways:

  • Under the @lamnhan/docsuper property of package.json file
  • The docsuper.config.js file for more advanced config
  • By the options param when init new docsuper(options?) instance.

Options properties

Name Type Description
apiGenerator? "none" | "typedoc" | CustomApiGenerator Detail API generator, default to 'typedoc'
cleanOutput? boolean Global clean output, can be overridden per file
converts? AdditionalConverts Additional converts
fileRender? BatchRender Output as standalone file
noAttr? boolean No generator footer attribution
srcPath? string Path to the source code, default to src
typedocConfigs? TypedocConfigs Custom Typedoc configs
url? string Docs homepage url, default to the Github Pages repo url
webRender? WebRender Output as a website
