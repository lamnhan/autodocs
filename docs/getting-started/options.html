Options

Options can be provided in 3 ways:

  • Under the @lamnhan/ayedocs property of package.json file
  • The ayedocs.config.js file for more advanced config
  • By the options param when init new ayedocs(options?) instance.

The Options interface

Complete module options

Name Type Description
cleanOutput? boolean Global clean output, can be overridden per file
fileRender? BatchRender Output as standalone file
refGenerator? "none" | "typedoc" | CustomReferenceGenerator Detail reference generator, default to 'typedoc'
srcPath? string Path to the source code, default to src
typedocConfigs? TypedocConfigs Custom Typedoc configs
url? string Docs homepage url, default to the Github Pages repo url
webRender? WebRender Output as a website

The WebRender interface

Specific config for render as a website

Name Type Description
categories? object Website article grouping
files BatchRender List of files
index? string Custom website index.html, default to a page that redirects to the first article
out? string Website output dir
theme? string Name of or path to a website theme, built-in themes: default
