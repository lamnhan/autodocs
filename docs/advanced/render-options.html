Render options

The RenderLocalOptions interface

Local options override any corresponding global options

RenderLocalOptions properties

Name Type Description
cleanOutput? undefined | false | true Ignore global sections (current content will be replaced)

The RenderWebOptions interface

Web options provides more information for a web page

RenderWebOptions properties

Name Type Description
deepMenu? undefined | false | true Show top level headings in the menu
pageTitle? undefined | string The page title
webData? WebData Data used in the webpage
