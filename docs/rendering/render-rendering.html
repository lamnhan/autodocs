Render rendering

Render direct rendering

To render a rendering (custom sections):

{
   'TEST.md': {
     section1: ['Main', 'SELF']
   },
}

Render rendering with options

You can render a rendering with options.

{ 'TEST.md': { rendering: { section1: ['Main', 'SELF'] }, // options } }
