Command overview

Document generator for Typescript projects.

  • ayedocs generate|g [path] --config [value] --package [value] --template [value]
  • ayedocs init|i --override
  • ayedocs preview|p [input] [output] [params...]
  • ayedocs show|s [input]
  • ayedocs help
  • ayedocs *

Command reference

generate

Generate the documentation.

Usage:

ayedocs generate [path] --config [value] --package [value] --template [value]
ayedocs g [path] --config [value] --package [value] --template [value]

Parameters:

  • [path]: Path to the output file

Options:

  • -c, --config [value]: Path to custom config file.
  • -p, --package [value]: Path to custom package file.
  • -t, --template [value]: Use this template for the [path] param.

init

Init Ayedocs for a project.

Usage:

ayedocs init --override
ayedocs i --override

Options:

  • -o, --override: Override existing configuration file.

preview

Preview a rendering.

Usage:

ayedocs preview [input] [output] [params...]
ayedocs p [input] [output] [params...]

Parameters:

  • [input]: The rendering input
  • [output]: The converting output
  • [params...]: The convert options

show

Show Declaration info.

Usage:

ayedocs show [input]
ayedocs s [input]

Parameters:

  • [input]: The rendering input

help

Display help.

Usage:

ayedocs help

*

Any other command is not suppoted.

Usage:

ayedocs <cmd>
