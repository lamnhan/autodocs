Options can be provided in 3 ways:

  • Under the @lamnhan/ayedocs property of package.json file
  • The .ayedocsrc.js file for more advanced config
  • By the options param when init new

    AyedocsModule

     instance.

Options list
Name Type Description
cleanOutput? undefined | false | true Global clean output, can be overridden per file
fileRender? BatchRender Output as standalone file
refGenerator? "none" | "typedoc" | CustomReferenceGenerator Detail reference generator, default to 'typedoc'
srcPath? undefined | string Path to the source code, default to src
typedocConfigs? TypedocConfigs Custom Typedoc configs
url? undefined | string Docs homepage url, default to the Github Pages repo url
webRender? WebRender Output as a website

Web options
Name Type Description
categories? undefined | object Website article grouping
files BatchRender List of files
index? undefined | string Custom website index.html, default to a page that redirects to the first article
out? undefined | string Website output dir
theme? undefined | string Name of or path to a website theme, built-in themes: default
